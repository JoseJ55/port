This project is named "Guess" a guessing game made with pygame and works with json files to get the question and answers. This was a simple project, but I didn't use any engine or help other than just small things when I got stuck. The main area I struggled in when making this program was when I was making the looks in pygame and changing them whenever there was a correct or wrong answer. I will update this program later or replace it with a newer more polish one.

Guess code

When running this code the first thing you will see is the main screen of the game. On the screen there is five buttons, one is the question button which gives you the question and answers in their respected box. The other four buttons are for the answers when you want to answer a question. Lastly, there is the box for the question to appear at.

When the question button is pressed the question and answers will appear in their respected box. The question will be a random question from the json file and appear in the box. The answers on the other hand will all be associated with that question, but they will be shuffled around every time the question is given so they might appear in a different box than before.

When one of the four question buttons is pressed the program checks if it is correct and turns the button red or green depending on the answer that was chosen.

When the answer is correct the button will turn green and the red one will turn back to normal. Even if the answers are shuffled the program will check if it is correct and turn the buttons to their respected color.
